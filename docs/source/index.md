# Oasis example models

Worked, executable examples for the Oasis example models — chiefly **PiWind**, the
reference toy model used across Oasis for demonstrations and CI.

These examples are **co-located with the model data** and kept honest by the repo's
model-run CI. They complement the engine/MDK documentation in the OasisLMF site
(which the aggregated Oasis documentation links to via intersphinx).
